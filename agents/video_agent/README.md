# video_agent

Cuts (Auto-Editor), assembles (FFmpeg), captions (renderer TBD by the
Phase 3.5 Hebrew RTL spike — see ADR-010), overlays B-roll/screenshots/
cursor highlights, and generates thumbnails. See `docs/roadmap.md` Phase
4.2–4.6 and `docs/open-source-landscape.md` section 4 for why several of
these sub-steps are custom code rather than an adopted library.
