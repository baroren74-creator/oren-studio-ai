# research_agent

Finds sources, reads documentation, summarizes, understands the
underlying technology, and finds the *original* source (not a reposted
copy). See `docs/agents.md` and `docs/roadmap.md` Phase 2.4–2.7 (incl.
the `interest_score` cost gate, ADR-003). Uses `providers/llm`,
`providers/crawl`, `providers/search`.
